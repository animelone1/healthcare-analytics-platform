# healthcare-analytics-platform
An end-to-end healthcare analytics project built using Python, MySQL and Power BI.  This project demonstrates the complete analytics workflow from raw healthcare data to executive dashboards.

The project demonstrates the complete data analytics workflow:

- Data Cleaning
- Data Validation
- Database Design
- SQL Analytics
- Dashboard Development
- Business Intelligence

The dataset consists of synthetic healthcare records generated by Synthea and includes patients, providers, encounters, claims, medications, procedures, observations, and organizations.

# Dataset

The project contains 18 relational healthcare tables.

| Table | Description |
|--------|-------------|
| patients | Patient demographics |
| providers | Healthcare providers |
| organizations | Hospitals and clinics |
| encounters | Patient visits |
| claims | Insurance claims |
| claims_transactions | Claim payments and adjustments |
| medications | Prescribed medications |
| procedures | Medical procedures |
| conditions | Diagnoses |
| allergies | Allergies |
| observations | Clinical observations |
| immunizations | Vaccinations |
| imaging_studies | Imaging records |
| devices | Medical devices |
| supplies | Medical supplies |
| careplans | Treatment plans |
| payers | Insurance companies |
| payer_transitions | Insurance enrollment history |

# Example SQL Query

```sql

DESCRIPTION,

COUNT(*) AS Cases

FROM conditions

GROUP BY DESCRIPTION

ORDER BY Cases DESC

LIMIT 10;
```
